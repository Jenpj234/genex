# GenEx – Generation Express
"Serving every generation in moments."

An e-commerce web app designed for senior citizens to easily shop for groceries, medications, and household essentials.

## Features
- Browse and search products
- Filter by category
- Add items to cart with live quantity updates
- View nearby pharmacies and hospitals

## Tech Stack
- React, Vite, React Router, useContext
